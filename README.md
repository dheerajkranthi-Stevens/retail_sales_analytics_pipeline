# Retail Sales Analytics Pipeline

## Overview
This project builds an end-to-end analytics pipeline to analyze retail sales performance and profitability using Python, SQL, and Power BI.

The workflow transforms raw sales data into business insights through data cleaning, SQL analysis, and dashboard visualization.

## Business Problem
Retail companies often struggle to understand:
- Which products drive profitability
- Whether revenue growth translates to profit growth
- Which regions underperform
- How discounts impact margins

This project analyzes sales transactions to identify revenue drivers and profitability trends.

## Tech Stack
- Python (Pandas)
- SQLite
- SQL
- Power BI
- GitHub

## Data Pipeline
Raw CSV → Data Cleaning → SQLite Database → SQL Queries → KPI Outputs → Power BI Dashboard

## Key Insights
- Revenue shows consistent growth with seasonal spikes.
- Profit margin fluctuates across months indicating discount effects.
- Technology category contributes highest profitability.
- Certain regions dominate sales performance.
- Sales growth does not always equal profit growth.
